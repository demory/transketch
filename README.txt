TRANSIT SKETCHPAD

** OVERVIEW **

Transit Sketchpad is an open source (GPLv3) vector illustation program intended to facilitate the creation of diagramatic transit maps.

** INSTALLATION **

To build the JAR file you will need the following:
 - Java 1.6: http://www.oracle.com/technetwork/java/javase/downloads/index.html
 - Ant: http://ant.apache.org/bindownload.cgi

A build file is included in the root directory of this repository. To build, cd to this directory and run 'ant'; to run use 'ant run'.
